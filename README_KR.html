# Chat Lobby SillyTavern을 위한 캐릭터 & 채팅 관리 확장 --- ## 주요 기능 ### 캐릭터 그리드 모든 캐릭터를 한눈에 확인. 썸네일 카드로 탐색하고, 호버 시 채팅 수와 메시지 수 표시. - 이름순 / 최근순 / 메시지순 정렬 - 태그 필터 ### 모바일 & 즐겨찾기 터치에 최적화된 반응형 UI. - 캐릭터 즐겨찾기 - 자주 쓰는 캐릭터 상단 고정 - 채팅 즐겨찾기 - 중요한 대화 따로 관리 ### 채팅 목록 & 폴더 캐릭터 클릭 시 채팅 목록 슬라이드 오픈. - 채팅 미리보기 (마지막 메시지) - 폴더로 채팅 분류 - 배치 선택으로 여러 채팅 한번에 이동 - 채팅 삭제 / 새 채팅 시작 ### 페르소나 & 검색 상단 바에서 페르소나 원클릭 전환. - 페르소나 바 휠 스크롤 지원 - 실시간 캐릭터 검색 - 태그 클릭 필터링 --- ## 메뉴 접근 | 환경 | 방법 | |------|------| | 데스크톱 | 우측 하단 FAB 버튼 | | 모바일 | 상단 햄버거 메뉴 (☰) | ### 햄버거 메뉴 기능 (모바일) - 🌓 테마 변경 - 🔄 새로고침 - 📅 캘린더 - 📊 Wrapped - 📥 봇 임포트 - 👤 페르소나 추가 - 🔧 디버그 ### 빠른 이동 - 채팅 목록 상단 봇 이미지 클릭 → 봇 카드 메뉴 - 페르소나 이미지 클릭 → 페르소나 메뉴 --- ## 추가 기능 ### 📊 Chat Wrapped 가장 많이 대화한 캐릭터, 총 메시지 수 등 다양한 통계 제공. ### 📅 캘린더 히트맵 GitHub 스타일 히트맵으로 대화 활동 시각화. ### 🎲 랜덤 캐릭터 주사위 버튼으로 랜덤 캐릭터 선택. --- ## 기술 - 🌓 다크 / 라이트 테마 - ⚡ 스마트 캐싱 - 🗂️ 로컬 데이터 매핑 (localStorage) - 📱 반응형 UI --- ## 설치 방법 1. SillyTavern 상단 **Extensions** 메뉴 열기 2. **Install Extension**에 아래 URL 입력 ``` https://github.com/gakumasu1717-cloud/SillyTavern_ChatLobby-dev-1 ``` 3. **Install** 클릭 후 새로고침 --- ## 이용 안내 - ✅ 깡통사이트 내 자유롭게 사용 가능 - ❌ 원작자 허락 없이 재배포 / 2차 공유 금지 - ❌ 외부 사이트 재배포 금지 - ❌ 상업적 이용 금지 --- © 2025 Chat Lobby.
